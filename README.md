## About
- 이 레포지토리는 시뮬레이션으로 생성, 처리 및 요약된 fMRI 신경활성화 데이터를 Python을 활용하여 통계분석한 내용을 담고 있습니다. 
- Paired T-Test, rmANOVA, Correlation, Regression, 등을 수행합니다.
- 연구에 대한 자세한 내용은 https://github.com/thoon4/consSTR_R 에서 확인할 수 있습니다.
